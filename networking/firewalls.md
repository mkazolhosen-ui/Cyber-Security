# Firewalls

## What is a Firewall?

- A firewall controls network traffic based on a set of rules.
- It can filter traffic by IP address, port, protocol, and application.

## Types of Firewalls

- Network firewalls protect traffic between networks.
- Host-based firewalls protect individual devices.
- Next-generation firewalls add application awareness and intrusion prevention.

## Cybersecurity Relevance

- Firewalls are a core defense in depth technology.
- They block unauthorized access and enforce segmentation.
- Firewalls help prevent threats from moving between network zones.

## Usefulness in Cybersecurity

- They provide visibility into network traffic and policy enforcement.
- Firewalls are critical for protecting web servers, remote access, and critical systems.
- Properly configured firewalls limit attack surface and support compliance.

## Best Practices

- Use deny-by-default rules and only allow required traffic.
- Keep firewall policies organized and regularly review them.
- Combine firewalls with IDS/IPS, VPNs, and monitoring for layered defenses.
