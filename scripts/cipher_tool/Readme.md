#  Caesar Cipher CLI Tool

A simple command-line tool for encrypting, decrypting, and brute-forcing Caesar cipher text.

---

##  Features

* Encrypt text
* Decrypt text
* Brute-force all possible shifts

---

##  Usage

### Encrypt

```bash
python cli.py -m encrypt -t "hello world" -s 3
```

### Decrypt

```bash
python cli.py -m decrypt -t "khoor zruog" -s 3
```

###  Brute Force

```bash
python cli.py -m brute -t "khoor zruog"
```

---

##  Example

```bash
Input:  Pbhentr vf sbhaq va hayvxryl cynprf.
Output: Courage is found in unlikely places.
```
